Instructions!

Welcome to the Backup File Discovery Challenge! Your objective is to locate and access a hidden backup file on the server.


  1. You will be navigating to a specific challenge page where a backup file is stored.

  2. In the URL, you'll find a parameter named file. Your task is to bruteforce this parameter with various file names to locate the correct backup file.

  3. The backup file follows the format <filename>.zip. Use tools like Burp Suite to capture requests and bruteforce the file parameter.

  4. Look for HTTP status code 200 OK in Burp Suite responses to identify successful attempts.

  5. Incorrect attempts will result in an HTTP status code 404 Not Found.

  6. The wordlist for brute-forcing backup files is provided. Click the button below to view it.
